# Contributing (Futuro Fortissimo book)

## Golden rule: citations

**Only cite what is includible:**
- **Corpus (FF.x.y)**
- **Notes** (only if the note is itself includible / internal)

### External links
If you encounter an **external source/link** in a chapter:
1) **Replace it with a better internal support** (preferred): add a short sentence supported by the corpus + an FF.x.y citation.
2) If you cannot replace it: **rewrite/remove** the claim.

Do **not** leave external citations as the only support.
